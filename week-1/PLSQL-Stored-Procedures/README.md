# Exercise 3: Stored Procedures

## Overview

This project demonstrates the implementation of PL/SQL Stored Procedures for a banking system using Oracle Database.

## Scenarios Implemented

### Scenario 1: Process Monthly Interest
- Created the `ProcessMonthlyInterest` stored procedure.
- Applied a 1% monthly interest rate to all savings accounts.
- Updated account balances automatically.

### Scenario 2: Update Employee Bonus
- Created the `UpdateEmployeeBonus` stored procedure.
- Accepted department name and bonus percentage as parameters.
- Updated employee salaries based on the specified bonus.

### Scenario 3: Transfer Funds
- Created the `TransferFunds` stored procedure.
- Checked whether the source account had sufficient balance.
- Transferred the specified amount from one account to another.
- Displayed a success or insufficient balance message.

## Technologies Used

- Oracle SQL
- PL/SQL
- Oracle Live SQL
