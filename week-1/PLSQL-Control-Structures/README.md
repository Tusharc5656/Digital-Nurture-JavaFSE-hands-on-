# PL/SQL Control Structures

## Objective
This project demonstrates the use of PL/SQL control structures such as loops, conditional statements, and updates using a simple banking database.

## Scenarios

1. Apply a 1% discount to loan interest rates for customers above 60 years of age.
2. Promote customers to VIP status if their balance exceeds 10000.
3. Print reminder messages for loans due within the next 30 days.

## Technologies Used
- Oracle SQL
- PL/SQL
- Oracle Live SQL
